# github-trending-cli
# GitHub Trending Repos

A simple CLI tool to fetch and display trending GitHub repositories in a nicely formatted table, filtered by time range, language, and result count.

## Features

- Fetch repositories created within the last day, week, month, or year
- Filter by programming language
- Limit the number of results
- Clean, colorized table output (stars, language, description, URL)

## Requirements

- Python 3.8+
- `requests`
- `rich`

## Installation

```bash
git clone <your-repo-url>
cd <your-repo-folder>
pip install -r requirements.txt
```
